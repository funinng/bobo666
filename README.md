# Vue 3 + TypeScript + Vite 项目

这是一个基于 Vue 3、TypeScript 和 Vite 的现代化前端项目，集成了 TailwindCSS 和多个 UI 组件。

## 技术栈

- **Vue 3** - 现代化的 Vue 框架
- **TypeScript** - 类型安全的 JavaScript 超集
- **Vite** - 快速的构建工具
- **TailwindCSS** - 实用优先的 CSS 框架
- **Pinia** - Vue 状态管理
- **Vue Router** - Vue 路由管理
- **Reka UI** - 无头 UI 组件库

## 项目结构

```
src/
├── components/          # Vue 组件
│   ├── ui/             # UI 组件库
│   └── ...
├── views/              # 页面组件
├── router/             # 路由配置
├── stores/             # Pinia 状态管理
└── ...
```

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产构建
npm run preview
```

了解更多关于推荐的项目设置和 IDE 支持，请查看 [Vue 文档 TypeScript 指南](https://vuejs.org/guide/typescript/overview.html#project-setup)。
